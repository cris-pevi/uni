\section{Ejemplo de Ecuaciones}
Resolver la siguiente ecuación cuadrática:

\[
ax^2 + bx + c = 0
\]

\section{Solución}
Usando la fórmula general:

\[
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
\]
